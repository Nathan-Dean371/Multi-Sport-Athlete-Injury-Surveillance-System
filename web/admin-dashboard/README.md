- Admin Dashboard
-
- A Next.js admin dashboard for the Multi-Sport Athlete Injury Surveillance System.
-
- ## Getting Started
-
- Install dependencies:
- ```bash

  ```
- npm install
- ```

  ```
-
- Run the development server:
- ```bash

  ```
- npm run dev
- ```

  ```
-
- Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
-
- ## Login Credentials
-
- For testing, use any email/password combination:
- - Email: admin@example.com
- - Password: any password
-
- ## Project Structure
-
- - `app/` - Next.js App Router pages and layouts
- - `page.tsx` - Login page (/)
- - `dashboard/page.tsx` - Dashboard home page (/dashboard)
- - `store/` - Zustand stores for state management
- - `styles/` - Global CSS styles
- - `public/` - Static assets
-
- ## Features
-
- - Dark theme with lime green accents
- - Login authentication flow
- - Dashboard with tour management
- - Responsive design
- - Built with Next.js 14, React 18, and Tailwind CSS
