# Coach Injury Reporting (Canonical)

## Summary

Coaches can report injuries through a guided flow with report type selection and player selection before form submission.

## Primary Flow

1. Select report type (quick or detailed)
2. Select team and player
3. Submit injury details

## Implementation References

- Mobile injury screens under `mobile/src/screens/injuries/`
- Navigation wiring in mobile navigation modules
